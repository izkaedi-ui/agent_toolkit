# Release Checklist

Before release:

- [ ] `npm run typecheck` passes.
- [ ] `npm run build` passes.
- [ ] `npm run orchestrate` passes.
- [ ] `npm run orchestrate:super` passes.
- [ ] `npm run lint:protocols` passes or only warns.
- [ ] `npm run self-test` passes.
- [ ] `npm run score` passes score threshold.
- [ ] `npm run validate:reports` passes.
- [ ] `npm run benchmark:arena` passes.
- [ ] Generated reports are attached as artifacts.
- [ ] Changelog updated.
- [ ] Version updated if publishing package.
- [ ] Limitations documented.

## Final Rule

No claim may exceed evidence.
