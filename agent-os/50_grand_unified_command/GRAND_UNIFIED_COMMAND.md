## Purpose

The Grand Unified Command is the top-level protocol activating ALL Agent OS gates simultaneously. Invoke this when maximum evidence gathering and validation is required.

## Activation

```

```

## What activates

When the Grand Unified Command is invoked, all of the following gates activate:

1. **Production Reality** (`20_production_reality`) — Error handling, environment config, dependency security, logging, build reproducibility, performance budget, accessibility, browser compatibility.

2. **Frontend Rendering Mastery** (`21_frontend_rendering_mastery`) — Render mode, hydration, streaming, Core Web Vitals, code splitting, image optimization, font loading, third-party impact.

3. **Design Systems** (`22_design_systems_visual_language`) — Design tokens, component consistency, dark mode, breakpoints, typography scale, spacing system, color contrast, motion tokens.

4. **3D Browser Bible** (`23_3d_browser_agent_bible`) — All 12 Laws of 3D Browser Development.

5. **Elite Animation Systems** (`24_elite_animation_systems`) — Real-state animation, reduced motion, 60fps budget, no layout thrash, honest progress.

6. **MCP Animation Toolkit** (`25_mcp_animation_toolkit`) — Full audit suite run via MCP tool calls.

7. **Animation Production Hardening** (`26_animation_production_hardening`) — MCP transport, browser extension diagnostics, large mesh fixtures, benchmark corpus, traces, WebGPU, shaders, physics, skeletal, timeline, perceptual quality, memory disposal, device compatibility, animated asset security.
